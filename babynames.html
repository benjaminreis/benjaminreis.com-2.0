What is this?

Anecdotal story: In choosing the names for our children, we wanted names that were unique, but still identifiable as names. When we named our oldest daughter "Harper" we knew of no other children with the same name. When our daughter was 2 we spent an evening at the zoo and found 4 other Harpers!

My wife and I have noticed that the name of each of our four children has more or less trended upwards in popularity since we chose the name. I wanted to chart the information to get a visual representation of this. I wanted to verify that this was more than just anecdotal, and to see if this carried over to our other childrens names.

How did I do this?

I used d3.js to display the chart. D3.js is a nifty JavaScript library for reading and displaying data. The data visualization possibilities using D3.js are vast. In my particular implementation, the line graph is drawn using Scalable Vector Graphics. It also allows utilizes CSS and HTML, but SVG draws a lot of neat stuff.

The data was obtained from Social Security Administration and their baby names database. I am reading the data from CSV files so it is easy to add and swap out datasets. I had to utilize a sister framework called "Queue.js" to enable reading in the 4 csv files in parallel. This allows for the processing of each files to happen asynchronously. In a large dataset this would be problematic to wait for incredibly large files
